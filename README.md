# Student Performance Tracker

## Overview
The **Student Performance Tracker** is a Python program that manages and organizes student academic records.  
It demonstrates the use of **searching, sorting, and hashing algorithms** to efficiently retrieve, rank, and store data.


## Features
- Add new student records using a hash table  
- Search students by ID or Name (Sequential & Binary Search)  
- Sort records using:
  - Bubble Sort  
  - Insertion Sort  
  - Merge Sort  
  - Quick Sort  
- Rank students using Heap Sort  
- Compare the time complexity of sorting algorithms  


## Data Structure
Each student record includes:
```python
StudentID
StudentName
Grade
CourseDetails
```
